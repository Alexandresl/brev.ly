# Brev.ly

![Ilustração Projeto](assets/Thumbnail.png)

Desafio da Pós Graduação da Faculdade de Tecnologia Rocketseat. Aplicação Full-Stack que permita o cadastro, listagem e remoção de links encurtados, geração de relatório dos acessos de cada link e também o redirecionamento correto do link encurtado para o link original.
